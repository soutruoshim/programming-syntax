Python Programming Cheatsheet

1. Basic Syntax

print("Hello, World!")

2. Variables and Data Types

TypeExample

3. Operators

Arithmetic: +, -, *, /, %, **, //

Comparison: ==, !=, >, <, >=, <=

Logical: and, or, not

Assignment: =, +=, -=, etc.

4. Control Flow

x = 5

if x > 0:
    print("Positive")
elif x == 0:
    print("Zero")
else:
    print("Negative")
for i in range(3):
    print(i)

j = 0
while j < 3:
    print(j)
    j += 1

5. Functions

def add(a, b):
    return a + b

print(add(3, 4))

6. Classes and Objects

class Samurai:
    def __init__(self, name, level):
        self.name = name
        self.level = level

    def introduce(self):
        print(f"I am {self.name}, level {self.level}")

7. Lists and Dictionaries

numbers = [1, 2, 3]
print(numbers[0])

ages = {"Alice": 25, "Bob": 30}
print(ages["Alice"])

8. File I/O

with open("file.txt", "w") as f:
    f.write("Hello File!")

with open("file.txt", "r") as f:
    text = f.read()
    print(text)

9. Exception Handling

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")

10. Modules and Imports

import math
