C Programming Cheat Sheet

Basic Structure

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Compiling and Running

gcc program.c -o program
./program

Data Types & Specifiers

TypeSpecifierExample
int%dint num = 5;
float%ffloat f = 5.5;
char%cchar c = 'A';
double%lfdouble d = 10.5;

Operators

int sum = a + b;
int mod = a % b;

Control Structures

if-else

if(score >= 60)
    printf("Pass");
else
    printf("Fail");

for-loop

for(int i = 0; i < 10; i++)
    printf("%d\n", i);

Arrays & Strings

int nums[3] = {1,2,3};
char str[] = "Hello";

Functions

int add(int x, int y) {
    return x + y;
}

Pointers

int x = 10;
int *ptr = &x;

Structures

struct student {
    int id;
    char name[20];
};

Dynamic Memory

int *arr = malloc(5 * sizeof(int));
free(arr);

File I/O

FILE *fp = fopen("file.txt", "w");
fprintf(fp, "Hello!");
fclose(fp);

Preprocessor Directives

#define MAX 100

Common Libraries

  • <stdio.h> - I/O functions
  • <stdlib.h> - Memory functions
  • <string.h> - String manipulation
  • <math.h> - Math operations

Best Practices

  • Initialize all variables.
  • Check return values of functions.
  • Free dynamically allocated memory.
  • Comment clearly and logically.
